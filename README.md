# Path Parameters and Query Parameters

A FastAPI learning project demonstrating path parameters and query parameters.

## Topics Covered

1. **Path Parameters and HTTP Methods** - Basic path parameters with HTTP methods
2. **Path Parameter Order Matters** - Understanding route order importance
3. **Path Parameter Predefined Values** - Using Enums for predefined path values

## Setup

```bash
cd "<folder-name>"
uv sync
uv run fastapi dev app/main.py
```

## API Docs

Visit `http://127.0.0.1:8000/docs` after running the server.
